## 🚀 Production-Grade Projects

### 🪐 [Stealing from Paradise](https://github.com/ldqanh1408)
> **A High-Concurrency Distributed Flash Sale Ecosystem**
- **Architecture:** Engineered a robust **9-module microservices infrastructure** featuring API Gateway, Identity, Order, Payment, Flash Sale, and Notification services.
- **Transaction Handling:** Integrated **Axon Framework** as the core transaction engine, executing transactional integrity through strict **CQRS and Event Sourcing** patterns.
- **Traffic Buffering:** Placed **Apache Kafka** as a high-throughput load buffer to gracefully absorb immense traffic spikes during flash-sale events.
- **Tech Stack:** Java 25, Spring Boot, Axon, Kafka, PostgreSQL, Redis.

### 📝 [AI-Integrated Smart Journal System](https://github.com/ldqanh1408)
> **Intelligent Journal Platform with Contextual RAG & Multi-Modal Analysis**
- **AI Orchestration:** Coordinated multiple state-of-the-art open-source local AI models including **LLaMA-cpp** (text processing), **BLIP** (image captioning), and **Faster-Whisper** (voice transcription).
- **Information Retrieval:** Built a custom **Retrieval-Augmented Generation (RAG)** pipeline leveraging **ChromaDB** vector storage for accurate personal knowledge querying.
- **Insights:** Executed deep **sentiment analysis** algorithms to track mental and emotional journal progression over time.
- **Tech Stack:** Python, LLaMA-cpp, ChromaDB, Spring AI, WebFlux.
